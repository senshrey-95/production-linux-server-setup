# 🚀 Production Linux Server Setup

A production-ready Ubuntu Linux server project demonstrating Linux administration, server hardening, automation, monitoring, Docker, Nginx, and system administration concepts.

---

## 📌 Project Overview

This project simulates the setup and administration of a production Linux server.

It demonstrates:

- Linux administration
- SSH hardening
- User and permission management
- UFW firewall configuration
- Fail2Ban protection
- Nginx web server deployment
- Docker & Docker Compose
- Systemd service creation
- Automated backups
- Log rotation
- System monitoring using Bash scripts
